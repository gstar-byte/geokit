# 2026-07-22

## GEOKit 站点 SEO/GEO 全面审计

对 geokit.site（Next.js App Router，Cloudflare 部署）做了代码级 + 线上验证的 SEO/GEO 审计。

**结论**：基础扎实（metadata、canonical、JSON-LD、llms.txt、AI 爬虫放行、PWA、安全头齐全），但未达"极致"。

**P0 问题**：
1. sitemap.xml 缺 3 个新工具页：/tools/geo-score、/tools/schema-validator、/tools/sitemap-validator（线上已验证）
2. /tools/sitemap-validator 无 layout.tsx → 继承首页 title（线上验证重复标题），无 description/canonical/JSON-LD
3. app/layout.tsx 有 hidden text div（display:none 塞 "swagger openapi graphql webmcp /api/mcp" 关键词），违反 Google 垃圾政策
4. 两个线上 404 引用：ai-plugin.json → /openapi.yaml（不存在）；head 中 link rel="mcp" → /api/mcp（不存在）

**P1 问题**：
- sitemap.ts 用 `new Date()` 生成 lastmod → 全站 lastmod 相同且不可信
- llms.txt / llms-full.txt 过期：写 11 个工具，实际 14 个；与首页 "Thirteen"、Organization JSON-LD 数据不一致
- About 页工具列表缺 3 个新工具内链
- /privacy、/terms 无 metadata → 重复首页标题
- About/Badge/Privacy/Terms 硬编码 text-white，默认浅色主题下标题不可见（UI bug）

**P2 优化空间**：robots.txt 可显式补 OAI-SearchBot/ChatGPT-User/Applebot-Extended/CCBot 等；可加 title template；工具页 FAQ 可加 FAQPage schema；中文 i18n 客户端切换对爬虫不可见（无 /zh 路由）。

用户只要求审计，未要求修复。修复清单已给出，待用户确认后执行。
